# Font Yamaha

Yamaha font is a monotype font.

## Example

Append `@niklauslee/font-yamaha` module.

```js
var font = require('@niklauslee/font-yamaha');
// get graphic context 'gc' from a driver (e.g. 'ssd1306')
gc.setFont(font);
gc.drawText(0, 0, "Lorem ipsum dolor sit amet...");
gc.display();
```
